# RunSmart Website

Website about choosing a heart rate monitor for running. This website is built with SCSS and JavaScript. It uses Slick (https://kenwheeler.github.io/slick/) and Wow.js (https://wowjs.uk/).

## Features

- Responsive design
- CSS animation
- Carousel image slider
- Tabs
- Form validation
